# Iris Regression – Linear Regression Model

A simple regression project using the classic **Iris dataset**.  
This project predicts **Sepal Width** based on **Sepal Length** using `LinearRegression`.

---

## Project Overview
This mini–machine learning project demonstrates:

- Loading the Iris dataset using scikit-learn  
- Selecting specific features  
- Training a Linear Regression model  
- Calculating MSE and R² score  
- Plotting Actual vs Predicted values  

It is a beginner-friendly example to understand basic regression and model evaluation.
